# Dataset Information

## Dataset Name

Heart Disease Dataset

## Description

The dataset contains patient clinical information used for predicting the likelihood of heart disease.

## Features

- Age
- Gender
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise Induced Angina
- Oldpeak
- ST Slope

## Target Variable

HeartDisease

0 → Low Risk

1 → High Risk

## Total Records

303

## Missing Values

Handled during preprocessing.
