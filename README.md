# Cardiac Arrest Prediction System

This project was developed as part of my learning in machine learning and web application development. It uses Python, Flask, and Scikit-learn to predict the risk of cardiac arrest based on patient health parameters. The application allows users to enter clinical data through a simple web interface and displays the predicted result using a trained machine learning model.

## Features

- Predicts cardiac arrest risk from patient input
- Flask-based web application
- Data preprocessing before prediction
- Machine learning model integration
- Simple and responsive user interface
- Displays prediction results instantly
- Model performance evaluated using standard metrics

## Technologies Used

- Flask

- CSS

## How to Run

1. Clone this repository.

```bash
git clone https://github.com/yourusername/Cardiac-Arrest-Prediction-System.git
```

2. Install the required packages.

```bash
pip install -r requirements.txt
```

3. Start the Flask application.

```bash
python app.py
```

4. Open your browser and visit:

```
http://127.0.0.1:5000
```
